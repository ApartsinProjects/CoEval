## Paper Report (HTML)

Generate a self-contained HTML report with all tables and figures:

```bash
python paper/gen_paper_report.py \
    --run benchmark/runs/paper-dual-track-v1 \
    --out paper/paper_report.html
```

The report includes Tables 3–9 (from `analysis.paper_tables`) and Figures 1–8 (Plotly interactive charts). Figures fall back to placeholder data with a visible watermark when benchmark scores have not yet been computed.
