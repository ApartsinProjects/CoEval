View 1 — Judge Ranking

About this table
What it shows: Each row is a judge model. Columns report inter-judge agreement scores averaged across all rubric aspects and all task–teacher–student combinations in the current filter.
SPA (Simple Percent Agreement) = fraction of pairwise evaluation pairs where both judges assigned the same score level. Range 0–1; higher is better.
WPA (Weighted Percent Agreement) = SPA weighted by the number of evaluation units — larger combinations contribute proportionally more.
κ (Kappa) = Cohen's κ; corrects for chance agreement. Interpretation: < 0 = worse than chance; 0–0.2 = slight; 0.2–0.4 = fair; 0.4–0.6 = moderate; 0.6–0.8 = substantial; > 0.8 = near-perfect.
⚠SJ flags self-judging (judge = student). ⚠ST flags self-teaching.

View 2 — Pairwise Agreement Matrix

About this figure
What it shows: A symmetric heatmap where each cell (i, j) shows the agreement between judge i and judge j on the same evaluation units. The diagonal is always 1.0 (a judge agrees with itself).
How to read it: High off-diagonal values (green) indicate that two judges produce consistent scores. Low values (red) suggest one judge is systematically more lenient or strict than another, or that the judges apply the rubric differently. Use the Agreement metric filter (SPA / WPA / Kappa) to switch the measure shown.

View 3 — Per-Aspect Agreement

About this figure
What it shows: Mean inter-judge agreement broken down by rubric aspect. Each bar represents one aspect; bar height = average agreement across all judge pairs for that aspect.
How to read it: Low agreement on a specific aspect means judges interpret that criterion inconsistently — the rubric description for that aspect may need clarification. High-variance aspects are less reliable for comparing student models.

About this figure
What it shows: A grouped bar chart of the fraction of High / Medium / Low scores awarded by each judge model across all evaluated responses.
How to read it: A judge that gives mostly High scores is lenient; one that gives mostly Low scores is strict. Large differences between judges indicate scoring bias. When using multiple judges, CoEval metrics are computed per-judge to avoid conflating genuine performance differences with judge calibration differences.
