View 1 — Judge Ranking

About this table
What it shows: Each row is a judge model with inter-judge agreement scores averaged across all rubric aspects and task–teacher–student combinations.
SPA (Simple Percent Agreement) = fraction of pairwise evaluation pairs where both judges assigned the same score level. Range 0–1; higher is better.
WPA (Weighted Percent Agreement) = SPA with partial credit for near-misses (High/Medium = 0.5 weight, High/Low = 0 weight).
κ (Kappa) = Cohen's κ; corrects for chance agreement. Interpretation: <0 = worse than chance; 0–0.2 = slight; 0.2–0.4 = fair; 0.4–0.6 = moderate; 0.6–0.8 = substantial; >0.8 = near-perfect.
Coverage-Weighted = same metric weighted by number of evaluation units per pair.
⚠SJ = self-judging (judge = student). Click any column to sort.

View 2 — Pairwise Agreement Matrix

About this figure
What it shows: A symmetric heatmap where each cell (i, j) shows the agreement between judge i (y-axis) and judge j (x-axis) on the same evaluation units. Both axes label the judge model name.
The diagonal is excluded (a judge trivially agrees with itself).
How to read it: Green = high agreement; red = low agreement / systematic bias. Use the Agreement metric filter (SPA / WPA / Kappa) to switch measures.
Note: Agreement can only be computed when two judges evaluated the same response–aspect pairs. If this heatmap shows no data, judges evaluated disjoint sets of responses (common in batch-split evaluation designs).
